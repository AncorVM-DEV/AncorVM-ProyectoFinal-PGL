# Umbra Knight

Umbra Knight es un juego de plataformas en 2D con mecánicas de acción y exploración inspirado en los géneros Metroidvania y Soulslike, desarrollado en Godot 4.

## 🗡️ Descripción del Juego
En este título controlo a un caballero atrapado en una mazmorra oscura. Mi objetivo principal es explorar el entorno para recolectar un total de 7 llaves que me permitirán escapar y alcanzar la victoria. 

El juego combina un plataformeo preciso con un combate táctico. Cuento con habilidades de movimiento como el salto, el deslizamiento por paredes y el rebote en las mismas (Wall Jump). También puedo realizar una voltereta (Roll) que me hace invulnerable durante unos instantes, ideal para esquivar ataques de enemigos o peligros del mapa.

## ⌨️ Controles
| Acción | Tecla |
| :--- | :--- |
| Moverse | A / D o Flechas |
| Saltar / Wall Jump | Espacio |
| Ataque con espada | Z / Clic Izquierdo |
| Voltereta (Roll) | Shift Izquierdo |
| Interactuar | E / Enter |
| Pausar Juego | Escape |

## 🚀 Características Principales
* **Sistema de Vidas:** Dispongo de 3 corazones. Al perderlos todos, la partida termina.
* **Knockback:** Al recibir daño de un enemigo, mi personaje sufre un retroceso físico.
* **Mecánica de Invulnerabilidad:** El "Roll" tiene un tiempo de recarga (cooldown) visible en la interfaz.
* **Exploración:** Niveles diseñados con verticalidad que requieren el uso del Wall Jump.
* **Enemigos:** Enfrentamientos contra esqueletos y murciélagos con patrones de movimiento específicos.
